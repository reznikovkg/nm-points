# Numerical methods - Points (decart system)

Last version: **v0.1** *(alpha)*

Date: **21.03.2018**

## Install

1) **Git clone** in your folder of the project
```git
git clone https://github.com/reznikovkg/nm-points.git
```

2) **Include** nm-points
```cpp
#include "nm-points/nm-points.h"
```
*Example including*
```cpp
#include "stdafx.h"
/* ... */
#include "nm-points/nm-points.h"
/* ... */
void main() {
/* ... */
}
```

3) Done!
